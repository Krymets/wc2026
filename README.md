# ⚽ Чемпіонат Світу 2026 — Розклад матчів

> Живий трекер матчів ЧС-2026 з автооновленням рахунків, голами, картками, статистикою гравців та сіткою плей-офф.

**Сайт:** [krymets.github.io/wc2026](https://krymets.github.io/wc2026/)

---

## Що вміє

- **Сьогодні** — матчі поточного дня + завтрашні з голами та картками по хвилинах
- **Розклад** — повний список усіх 104 матчів з подіями кожного матчу
- **Групи** — живі таблиці 12 груп із підрахунком очок, різниці голів і зонами виходу
- **Плей-офф** — сітка від 1/16 фіналу до фіналу
- **Статистика** — бомбардири, асисти, жовті/червоні картки, розподіл голів по хвилинах
- **Live-рахунки** — автооновлення кожну хвилину через ESPN API
- **Годинник Київ** — час відображається у часовому поясі UTC+3

## Технології

- Чистий HTML + CSS + Vanilla JS — без фреймворків і залежностей
- Один файл `index.html`, готовий до хостингу де завгодно
- ESPN Scoreboard API для live-рахунків та подій матчів
- `localStorage` для кешування подій між сесіями
